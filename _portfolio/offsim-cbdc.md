OffSim-CBDC is an agent-based simulation platform designed to study offline Central Bank Digital Currency (CBDC) ecosystems. The simulator models users, merchants, transactions, synchronization mechanisms, and fraud scenarios in disconnected environments.

Key features include:

* Offline transaction simulation
* Double-spending attack scenarios
* Merchant behavior modeling
* Fraud detection datasets generation
* Machine learning integration
* Performance and resilience evaluation

**Technologies:** Python, Mesa, Machine Learning, CBDC.
